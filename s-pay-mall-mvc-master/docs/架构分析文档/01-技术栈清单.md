# 技术栈清单

## 一、后端技术栈

### 核心框架
- **Spring Boot**: `2.7.12` - 核心应用框架
- **Spring Web**: 提供 RESTful API 支持
- **Spring Scheduling**: 定时任务支持（@EnableScheduling）

### 数据持久层
- **MyBatis**: `2.1.4` - ORM 框架
- **MySQL Connector**: `8.0.23` - MySQL 8.x JDBC 驱动

### 工具库
- **Lombok**: 简化 Java Bean 代码
- **Apache Commons Lang3**: `3.9` - 工具类库
- **Google Guava**: `32.1.3-jre` - Google 工具库（缓存、集合等）
- **FastJSON**: `2.0.28` - JSON 序列化/反序列化（阿里巴巴）
- **Commons Codec**: `1.15` - 编码/解码工具

### 网络通信
- **Retrofit2**: `2.9.0` - HTTP 客户端（用于调用微信 API）
  - converter-jackson: Jackson 转换器
  - adapter-rxjava2: RxJava2 适配器

### 第三方支付
- **支付宝 SDK**: `4.38.157.ALL` - 支付宝沙箱支付集成

### 身份认证
- **JWT (jjwt)**: `0.9.1` - JWT 令牌处理
- **Java-JWT (auth0)**: `4.4.0` - JWT 工具库

### XML 处理
- **dom4j**: `1.6.1` - XML 解析
- **XStream**: `1.4.18` - XML/对象转换

### 缓存（可选）
- **Redisson**: `3.26.0` - Redis 客户端（已注释，未启用）

### 事件总线
- **Google Guava EventBus**: 事件发布订阅机制

## 二、数据库

- **MySQL**: `8.0.x` - 关系型数据库
- **字符集**: UTF8MB4
- **存储引擎**: InnoDB

## 三、开发工具链

### 构建工具
- **Maven**: 项目构建和依赖管理
- **Java 版本**: `1.8` (JDK 8)

### 日志框架
- **Logback**: Spring Boot 默认日志框架
- **SLF4J**: 日志门面

### 开发环境
- **Maven 仓库**: 阿里云镜像 (http://maven.aliyun.com/nexus/content/groups/public/)

## 四、部署工具

### 容器化
- **Docker**: 支持 Docker 部署
- **Docker Compose**: 多容器编排

### 反向代理
- **Nginx**: Web 服务器和反向代理

### 内网穿透
- **Natapp**: 内网穿透工具（开发调试用）

## 五、第三方服务集成

### 微信生态
- **微信公众号**: 扫码登录、消息推送
- **微信开放平台**: 获取 AccessToken、生成二维码

### 支付服务
- **支付宝沙箱环境**: 支付功能测试

## 六、项目特性

- **多环境配置**: 支持 dev、test、prod 环境
- **定时任务**: 订单超时关闭、支付回调检测
- **事件驱动**: 支付成功后的事件通知机制
- **统一响应格式**: Response 统一封装
- **异常处理**: 自定义异常 AppException
- **跨域支持**: @CrossOrigin 注解支持跨域请求

## 七、技术架构模式

- **MVC 分层架构**: Controller → Service → Dao
- **领域驱动设计**: Domain 层独立（req/res/vo/po）
- **模块化设计**: Maven 多模块项目（common、dao、domain、service、web）
- **依赖注入**: Spring IoC 容器管理
- **AOP 支持**: Spring AOP（通过注解）
